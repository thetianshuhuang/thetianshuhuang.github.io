LC-3b Microarchitecture Poster

A gate-level (where possible, at least) depiction of the LC-3b microarchitecture, as used by Yale Patt's Computer Architecture class (EE 460N) at UT Austin.

The various diagrams specifying the datapath and microarchitecture are compiled into one massive poster, and monolithic data path components (Register File, ALU, etc) are elaborated into specific gate-level implementations.

This project was mostly an excuse to properly learn how to use Adobe Illustrator.
